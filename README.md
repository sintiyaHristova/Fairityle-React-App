## Fairityle 

Share your author's fairy tales and describe your favorite characters from them. 

## Deployment 

https://fairityle-sintiya.netlify.app/ 
 
## Front-end 

-   ReactJS 
-   JavaScript 
-   HTML 
-   CSS  

## Back-end 

-  Firebase  

## Usage 

In the project directory, you can run: 

-  Npm install 

-  Npm run build 

-  Npm start 

## Features 

# Public 

-  Home page 

-  Login 

-  Register 

-  Catalogue     

-  Details (but not to the functional activities)

# Private 

-  Logout 

-  Home page   

-  Create 

-  Catalogue   

-  Edit 

-  Details 

-  Delete
